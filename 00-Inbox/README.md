# Inbox-收集箱

临时捕获区。新想法、新概念先丢这里，24小时内处理。

## 使用方法

1. 快速记录：有新想法立即写进来
2. 触发流转：输入 `/sort-my-notes` 让 Claudian 帮助整理
3. 清空：处理完成后删除内容

## 🛠️ UE 笔记流水线（记不住就看这）

**口诀：校词 → 出大纲 → 整理。** 按顺序用这三个斜杠命令：

| 顺序 | 命令 | 干什么 |
|---|---|---|
| ① | `/fix-ue-terms` | 校正视频转写里听错的术语（义务→异步、降谱→Launch…） |
| ② | `/lesson-outline` | 转写 → 课程大纲，并把"值得记的"丢进本 Inbox |
| ③ | `/sort-my-notes` | 把 Inbox 蒸馏成原子笔记 + 回填大纲链接 + 写当日日记 |

> - **看视频学的** → ① → ② → ③ 走完整流水线。
> - **自己手敲/截图丢进来的** → 直接 ③。
