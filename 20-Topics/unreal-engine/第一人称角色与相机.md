# 第一人称角色与相机

## 一句话定义

> [!abstract]
> 第一人称角色的"业内传统做法"：==用两套模型==（自己看的 / 别人看的）+ 相机挂在头部 Socket。这样既能低头看到自己身体，别人看你也正常。

## 📖 要点

### 双模型（核心）

| 模型 | 设置 | 给谁看 |
|---|---|---|
| **自己看的** | `Only Owner See` ✅ + `First Person Primitive` + **关阴影** | 第一人称视角（头/手位置经动画特调，避免低头穿帮） |
| **别人看的** | `Owner No See` ✅ | 正常第三人称外观（用它投阴影） |

- 为什么两套：第一人称低头若用正常模型会看到诡异的悬空身体；所以"自己看的"那套用动画把头手调到合适位置，"别人看的"保持正常。
- 阴影：自己看的模型阴影怪 → **关它的 Cast Shadow**，用别人看的那套投影。

### 相机

- 挂到 Skeletal Mesh 的 **`head` Socket**（见 [[骨骼与 Skeletal Mesh]]）→ 跟着头动。
- **FOV 视野角**：大 = 广角（物体显小、视野宽）；小 = 长焦（拉近、物体显大）。
- `Camera Mesh Hidden in Game` 取消勾选 → 游戏里能看到相机模型，方便调试。
- 没挂相机时引擎会默认给一个；挂了就用你挂的。
