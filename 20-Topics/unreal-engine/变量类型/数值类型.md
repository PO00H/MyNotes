# 数值类型（Integer / Float / Byte）

## 一句话定义

> [!abstract]
> 数值类型 = ==装数字、能做算数==的变量。按"要不要小数、范围多大"分：整数用 Integer，小数用 Float，0–255 的小范围用 Byte。

## 📖 要点

| 类型 | 装什么 | 典型用途 |
|---|---|---|
| **Integer** 整数 | 没有小数点（…-1, 0, 1, 2…） | 计数、索引、分数、数量 |
| **Float** 浮点 | ==带小数==（3.14、0.5） | 距离、时间、速度、比例 |
| **Byte** | 0–255 的小整数 | 枚举底层、颜色通道、省内存 |

- ==选择口诀==：要小数就 Float，纯计数就 Integer，超小范围/枚举才用 Byte。
- **整数除法有坑**：`5 / 2`（两个 int）结果是 `2`（截断小数）；想要 `2.5` 得用 Float。
- UE5 里 Float 蓝图引脚常显示为 **Real**（底层 double），用法不变。

> [!warning] 易错点
> - **该用 Float 却用了 Integer** → 小数被砍掉，结果不对（如算平均值）。
> - **类型不匹配连不上** → 蓝图会提示插入转换节点（int → float 自动可转）。
