## 📖 教学（核心原理）

### 1. 为什么要分类型

- ==类型 = 给数据贴标签==，告诉引擎"这块内存装的是什么、能做什么运算"。
- 蓝图用==颜色==区分类型（红=bool、绿=string、金=struct…），引脚颜色对不上就连不起来——天然防错。

### 2. 怎么挑类型

- **真假开关** → 布尔；**算数/计数** → 数值；**给玩家看的字** → 字符串（FText）。
- **一堆相关的数**（位置 XYZ、旋转）→ 结构体，别用三个 float 散着传。
- **一串同类的东西**（敌人列表、物品栏）→ 容器（Array）。

### 3. 标量 vs 结构体 vs 容器

- ==标量==：单个值（bool / int / float）。
- ==结构体==：几个值打包成一个整体（Vector = X+Y+Z），可用 [[蓝图引脚拆分与合并]] 拆装。
- ==容器==：同一类型的多个值排成一列（Array<int>）。
