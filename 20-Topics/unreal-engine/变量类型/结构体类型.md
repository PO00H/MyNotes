# 结构体类型（Vector / Rotator / Transform）

## 一句话定义

> [!abstract]
> 结构体 = ==把几个相关的数打包成一个整体==。UE 里最高频的三个：Vector（位置/方向）、Rotator（旋转）、Transform（位置+旋转+缩放三合一）。每个 Actor 时刻都在用它们。

## 📖 要点

| 结构体 | 由什么组成 | 表示 |
|---|---|---|
| **Vector** | X / Y / Z（三个 Float） | 位置、方向、速度 |
| **Rotator** | Pitch / Yaw / Roll | 朝向 / 旋转角度 |
| **Transform** | Location + Rotation + Scale | 一个物体在世界里的完整摆放 |

- ==为什么打包==：位置就是要 XYZ 一起才有意义，散成三个 Float 传又乱又易错。
- **拆装**：用 [[蓝图引脚拆分与合并]]——Break 拆出 X/Y/Z，Make 把分量拼回 Vector。
- **Transform 是套娃**：它内部就装着一个 Vector（位置）+ 一个 Rotator（旋转）+ 一个 Vector（缩放）。
- 关联 [[Viewport]]：视口里 W/E/R 调的"移动/旋转/缩放"，改的就是选中物体的 Transform。

> [!warning] 易错点
> - **Vector 当方向用时忘了归一化** → 方向向量通常要 Normalize 成长度 1。
> - **Rotator 三个轴搞混** → Pitch 抬头、Yaw 转身、Roll 翻滚。
