# 枚举 Enum

## 一句话定义

> [!abstract]
> 枚举（Enumeration）= ==一组起好名字的固定选项，只能取其中之一==。比如角色状态：待机 / 巡逻 / 攻击。用它代替"记数字"，可读性和安全性都高得多。

## 📖 要点

![[Pasted image 20260604101204.png]]

- **本质**：把"0/1/2"这种 magic number 换成有名字的选项（Idle / Patrol / Attack），底层其实是 Byte。
- **建法**：内容浏览器右键 → 蓝图 → **枚举(Enumeration)** → 列出各选项。
- **怎么用**：
	- 建一个该枚举类型的变量存当前状态。
	- 用 **`Switch on Enum`** 节点：按当前枚举值**自动分出多条执行线**（比一堆 [[布尔 Boolean\|bool]] + Branch 干净得多）。
- **最常见场景**：==状态机==（角色/AI 的状态切换）、类型标识（武器类型、伤害类型）。

> [!tip] 枚举 vs 布尔 vs 数组
> - 只有两种情况（开/关）→ [[布尔 Boolean]]
> - ==三种以上互斥情况==（待机/巡逻/攻击）→ **枚举**
> - 一串同类数据（敌人列表）→ [[容器类型\|数组]]

> [!warning] 易错点
> - **用 int 当状态**（0=待机…）→ 难读易错；改用枚举 + Switch on Enum。
> - **漏处理某个选项** → Switch on Enum 会列出所有分支，别漏接。
