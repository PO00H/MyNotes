# 动画蓝图

## 一句话定义

> [!abstract]
> 动画蓝图（Animation Blueprint）= 驱动骨骼播放动画的蓝图。它分两块：==**EventGraph 取数据/算状态** + **AnimGraph 根据数据输出动作**==。

## 📖 要点

- **创建**：内容浏览器右键 → Animation → Animation Blueprint → **选骨骼**（动画绑骨骼）。
- **两图分工**（关键）：
	- **EventGraph**：和普通蓝图一样，用来**算"现在该播什么"**——取速度、是否下落，存成变量。
	- **AnimGraph**：没有执行流，**根据变量输出最终动作**（接 Blend Pose / 状态机 → 小人）。
- **取"谁在用这个动画"**：
	- `TryGetPawnOwner` 或 `GetOwningActor` → `Cast` 成角色 → 存下 `CharacterMovement`。
	- ==只在初始化（BeginPlay/Initialize）Cast 一次==，别在 Update 每帧 Cast（费）。
- **isMoving 判断**：取**水平速度** `Vector Length XY` > 阈值（如 0.01）。用 XY 是因为跳跃时垂直速度不算"移动"。
- **isFalling 判断**：`CharacterMovement → IsFalling`（在空中=true）。

![[Pasted image 20260608214604.png]]

> [!warning] 易错点
> - **每帧 Cast** → 性能差；引用类一次性存好。
> - **用整体速度判断移动** → 跳起来会误判在走；要用**水平**速度（XY）。
> - 指定动画蓝图：在 Mesh 的 `Use Animation Blueprint` 里挂上，否则不生效。
