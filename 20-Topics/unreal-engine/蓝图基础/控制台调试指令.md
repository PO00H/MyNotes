# 控制台调试指令

## 一句话定义

> [!abstract]
> 运行时在控制台（按 `~` 键，或编辑器底部 `Cmd` 框）输入的==调试命令==，用来临时显示/切换引擎信息，排查问题用。

## 📖 常用

| 指令 / 快捷键 | 作用 |
|---|---|
| `show collision` | 切换显示==碰撞体==（看触发盒/碰撞形状） |
| `stat fps` | 显示帧率 |
| `stat unit` | 显示帧耗时（Game/Render/GPU） |
| `Shift + F1` | 试玩(PIE)中==找回鼠标指针==（脱离游戏输入） |

> `show` 是一族"显示标志"指令（show + 名字），切换各种调试可视化；和视口的 Show Flags 同源（见 [[Viewport]]）。

> [!tip] 排查碰撞必用
> 做 [[碰撞 Collision]] / 触发器时，先 `show collision` 把看不见的碰撞体显形——很多"没触发"的 bug 一眼就看出是碰撞体没对上。
