# 蓝图引脚拆分与合并

## 一句话定义

> [!abstract]
> 蓝图节点的引脚可以==拆开看分量==、也可以==合并回整体==。处理结构体（如 Vector 的 X/Y/Z、Rotator）时极常用——不必每次都连一个 Make/Break 节点。

## 📖 要点

> 节点惯例：==左边是输入，右边是输出。==

| 操作 | 怎么做 | 效果 |
|---|---|---|
| **拆分 Split** | 引脚上右键 → `Split Struct Pin` | 一个引脚展开成多个分量（X / Y / Z…） |
| **拆分 Break** | 从==出口==引脚拖出 → `Break ...` | 生成一个拆解节点，把结构体拆成分量 |
| **合并 ReCombine** | 已拆开的引脚右键 → `Recombine` | 把展开的分量收回成一个引脚 |
| **合并 Make** | 从==入口==引脚拖出 → `Make ...` | 生成一个组装节点，把分量拼成结构体 |

- ==记法==：Split/Recombine 是"原地展开/收起"引脚；Break/Make 是"另起一个节点"拆/装。
- 适用对象：任何结构体引脚（Vector、Rotator、Transform、自定义 Struct）。
