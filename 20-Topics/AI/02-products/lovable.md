# Lovable-AI应用开发平台

## 一句话定义

Lovable（原名 GitWit），AI 驱动的全栈应用开发平台，通过自然语言描述即可生成完整的 Web 应用程序。

## 核心原理

- **自然语言编程** - 用中文/英文描述需求，AI 生成代码
- **可视化编辑** - 生成的应用可拖拽调整 UI
- **GitHub 集成** - 自动生成代码仓库，可导出代码
- **全栈生成** - 前端 + 后端 + 数据库 一站式生成

## 与 AI 的关系

Lovable 是 [[AI]] 在**软件开发领域**的应用：

- 将 [[../04-concepts/llm|LLM]] 的代码能力产品化
- 让非程序员也能开发应用
- 实现"想法到产品"的最短路径

## 产品定位

| 维度 | 定位 |
|------|------|
| **对标** | v0（Vercel）、Replit Agent、Bolt |
| **优势** | 编辑体验好、代码质量高、GitHub 集成深 |
| **适用** | MVP 开发、原型设计、内部工具 |

- 快速搭建产品 Landing Page
