# Manus-AI Agent产品

## 一句话定义

Manus（由 Monica.im 团队开发），通用型 AI Agent，能够独立思考、规划并执行复杂任务，直接交付完整成果。

## 核心原理

- **多 Agent 协作** - 多个专门 Agent 分工处理不同子任务
- **工具链集成** - 浏览器、代码编辑器、文件系统等工具
- **记忆与状态** - 保持长期记忆，跨会话连续工作
- **异步执行** - 任务在云端运行，无需实时等待

## 与 AI 的关系

Manus 是 AI **Agent 技术**的集大成者：

- 超越 [[../04-concepts/llm|LLM]] 的对话能力，实现**端到端任务交付**
- 代表 AI 从**被动响应**到**主动执行**的跃迁
- 是通往 AGI 的重要里程碑

## 产品特点

| 特点 | 说明 |
|------|------|
| **通用性** | 不限定领域，任何知识工作都能处理 |
| **自主性** | 接到任务后自主规划执行，无需逐步指令 |
| **交付物** | 直接输出文档、代码、报告等成果 |
| **异步** | 关闭电脑后任务继续执行 |

## 典型应用场景

- 深度研究并生成报告
- 编写并部署完整应用
- 数据分析与可视化
- 旅行规划与预订
- 简历筛选与评估
