# Memory-记忆系统

## 一句话定义

Claude Code 的持久化记忆系统，用于存储和检索跨会话的知识、上下文和用户偏好，让 AI 能够"记住"之前的交互。

## 核心原理

- **向量存储** - 使用嵌入向量进行语义检索
- **图结构** - 实体关系网络，支持复杂查询
- **分层缓存** - 短期（会话内）+ 长期（跨会话）记忆
- **自动总结** - 压缩和提炼历史信息

## 与 AI 的关系

Memory 解决了 [[llm]] 的核心局限：**上下文窗口有限**

- 原生 LLM 只能记住当前对话（几千到几万 token）
- Memory 系统让 AI 拥有"长期记忆"
- 实现真正的[[个性化AI]] - 越用越懂你

## 连接

## 应用场景

| 场景 | Memory 作用 |
|------|-------------|
| 跨会话项目 | 记住上周讨论的需求 |
| 个人偏好 | 记住你喜欢的代码风格 |
| 知识积累 | 从对话中提取模式保存 |
| 错误避免 | 记住之前犯过的错误 |
