## Claude Code 生态

Claude Code 的核心组件：

- [[harness]] - 运行时框架，系统架构基础设施
- [[memory]] - 记忆系统，持久化知识管理
- [[plugins]] - 插件系统，能力扩展机制
- [[skills]] - 技能系统，专业功能模块

### 关系图解

```
        [Harness] ← 运行时框架
       /    |    \
      /     |     \
[Plugins] [Memory] [Skills]
    |        |        |
  扩展     记忆      专业能力
```
